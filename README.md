# Proyecto Git

Este proyecto contiene un programa en Java que incluye las siguientes funcionalidades:

-Imprime "Hola mundo"
-Muestra la fecha y hora acual
-Permite crear usuarios introduciendo nombre, apellidos y emali
-Guarda los usuarios en un array y los muestra por pantalla

# Estructura del proyecto

-Principal.java: contiene el programa principal y la logica
-Usuario.java: define la clase Usuario

# Ramas utilizadas 

-main: version estable del proyecto
-Practica4_1: añade la funcionalidad de fecha y hora
-Practica4_1B: añade gestion de usuarios
